| No. | Experiment Title | Aim |
|-----|------------------|-----|
| **1** | Illustrate class & objects | To study and illustrate the concept of classes and objects in C++. |

| **3** | Function Overloading | To demonstrate the concept of function overloading applied to member functions. |
| **4** | Passing and returning objects | To implement passing objects as function arguments and returning objects from a function. |

| **11** | Binary operator overloading | To implement the use of binary operator overloading. |
| **12A** | Assignment operator overloading | To implement assignment operator (`operator=`) overloading for user-defined classes. |

| **17** | Pure virtual function | To implement pure virtual functions and abstract classes. |

| **19** | Function template | To implement and demonstrate the use of function templates. |
| **20** | File handling | To demonstrate file handling operations (read and write). |
| **21** | Exception handling | To implement exception handling using `try`, `throw`, and `catch`. |
| **22** | User-defined namespace | To implement the use of user-defined namespace in C++. |
| **23** | Signal (interrupts) handling | To implement signal handling in C++. |
| **24** | Simple programs using C++ STL | To implement simple programs using the Standard Template Library (STL). |
| **25** | Web programming with C++ | To demonstrate basic web programming integration using C++. |
| **26** | Graphics concepts in C++ | To implement simple programs using basic graphics concepts in C++. |
